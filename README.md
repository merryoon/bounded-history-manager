Bounded History Manager
This project implements a bounded undo-history system using a Double-Ended Queue (Deque).
 Features
- Stores only the last N actions
- Automatically removes the oldest action when capacity is full
- Supports undo functionality
- Command-line interface (CLI)
 Data Structure Used
- Deque (from Python collections)
  - Efficient insertion at back
  - Efficient removal from front
 How to Run
1. Clone the repository
2. Navigate to the folder
3. Run the program
